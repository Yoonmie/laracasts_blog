## Installation

```
- git clone https://github.com/Yoonmie/laracasts_blog.git
- cd laracasts_blog
- composer install
- cp .env.example .env
- create database
- php artisan key:generate
- php artisan migrate
- php artisan serve
```

## Project Flow 
- Can access all the posts
- Can access individual title
- Can access individual category


## Post
